# 頤安本草 · 新站改進與廣告策略簡報

11 頁 HTML 幻灯片，品牌色 paper / pine / cinnabar。

## 開啟方式

1. 用瀏覽器打開 `index.html`（雙擊或拖入 Chrome / Edge）
2. 預設進入 **3D 概覽牆** — 點任意卡片或右下角「▶ 开始演示」
3. 演示模式：`←` `→` / 空白鍵翻頁 · `ESC` 回概覽 · `P` 列印
