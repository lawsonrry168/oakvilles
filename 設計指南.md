## 1. 設計目標與原則

### 1.1 核心目標

| 目標 | 衡量指標 |
|------|----------|
| 將網站訪客轉化為預約病人 | 預約點擊率 > 8% |
| 建立醫師專業信任感 | 頁面停留時間 > 90秒 |
| 提升本地搜尋排名 | Google「附近中醫」前5位 |
| 提供清晰的服務資訊 | 跳出率 < 55% |

### 1.2 設計原則

**1. 信任優先（Trust First）**
每個頁面的設計決策都應以「如何讓初次訪客信任這位醫師」為首要考量。中醫診所的核心競爭力是信任，而非價格。

**2. 行動導向（Action Oriented）**
每個版塊必須引導用戶採取下一步行動——閱讀更多、WhatsApp 預約、查看地址。不留任何「死胡同」。

**3. 內容為王（Content Driven）**
以病症為中心的內容架構，讓用戶能快速找到自己關心的問題，同時驅動 SEO 自然流量。

**4. 行動裝置優先（Mobile First）**
香港用戶超過 75% 以手機搜尋本地服務。所有設計從 375px 開始，向上延伸。

**5. 文化共鳴（Cultural Resonance）**
設計語言融合傳統中醫美學（沉穩、自然、療癒）與現代專業感，不落入俗套的「古舊中藥鋪」視覺。

## 2. 目標受眾分析

### 2.1 主要受眾群

#### 群體 A：主動症狀搜尋者（佔 55%）
- **特徵：** 25–55歲，有明確病症，如痛症、皮膚問題、婦科困擾
- **搜尋行為：** Google「[症狀] 中醫 [地區]」
- **痛點：** 不確定哪位醫師值得信任，擔心療效與收費
- **設計回應：** 病症專頁 + 醫師資歷 + 透明收費

#### 群體 B：慢性病調理者（佔 25%）
- **特徵：** 35–65歲，已試過西醫，尋求長期調理
- **搜尋行為：** 「中醫調理身體」「中醫體質分析」
- **痛點：** 希望有系統性的治療方案，非單次診症
- **設計回應：** 療程介紹 + 案例說明 + 預約流程透明化

#### 群體 C：口碑轉介者（佔 20%）
- **特徵：** 各年齡層，由親友介紹
- **搜尋行為：** 直接搜尋醫師名字「伍厚臻」
- **痛點：** 需要確認聯絡方式與診症時間
- **設計回應：** 首頁明顯的聯絡資料 + 地址地圖

### 2.2 用戶旅程地圖

```
搜尋症狀
    ↓
找到網站（Google / 社交媒體）
    ↓
首頁：確認這是相關診所（5秒測試）
    ↓
服務頁：了解能否治療我的問題
    ↓
醫師頁：確認醫師資歷可信
    ↓
收費/流程頁：了解費用與過程
    ↓
聯絡/預約：WhatsApp 或電話預約
```

## 3. 品牌定位

### 3.1 品牌個性

| 維度 | 定位 |
|------|------|
| 專業度 | 嚴謹、有憑有據的醫學專業 |
| 溫度 | 溫暖、有耐心、以人為本 |
| 現代感 | 當代美學，非守舊傳統 |
| 在地感 | 了解香港病人需求 |

### 3.2 核心價值主張

**主標語（Hero Tagline）：**
> 「根本調理，恢復你的自然平衡」

**副標語選項：**
- 「伍厚臻中醫師 — 專業診療，用心調理」
- 「以中醫智慧，解決現代都市病」

### 3.3 品牌聲音（Brand Voice）

- **語氣：** 專業但平易近人，避免過度醫學術語
- **稱謂：** 「您」而非「你」，體現尊重
- **句式：** 短句為主，清晰直接，不用大量修飾語
- **避免：** 誇大療效、絕對性保證、貶低西醫

#### 強調色（Accent Palette）

```
金褐色      #B7881C    ████  傳統中醫感、特色標籤、分隔線裝飾

#### 色彩使用規則

- 主綠色 `#2D6A4F` 作為主要 CTA 按鈕、連結、導航選中狀態
- 頁面背景用 `#F5F5F0`（非純白），減少視覺疲勞
- 金褐色 `#B7881C` 僅用於裝飾性元素，切勿用於大面積背景
- 文字對比比例需達 WCAG AA 標準（4.5:1 以上）

### 4.2 字體系統

#### 字體選擇

```
標題字體：  思源宋體 (Noto Serif TC)
            — 傳遞專業、人文、可信感
            — 粗細：700（主標）、600（副標）、400（引文）

內文字體：  思源黑體 (Noto Sans TC)
            — 清晰易讀，現代感
            — 粗細：400（正文）、500（強調）、600（小標）

數字/英文：Inter
            — 用於電話號碼、地址門牌、英文標籤
```

```

### 4.6 圖像風格指南

**攝影方向：**
- 醫師職業照：白衣或中式醫師服，淺色背景，親切自然表情
- 診所環境：乾淨明亮，展示針灸、草藥等真實診療場景
- 療程圖示：局部特寫，展示專業操作（針灸、推拿）
- 避免：過度修圖、素材庫假圖、恐怖的病症特寫

**插圖風格（如無真實照片）：**
- 線條風格插圖，主綠色 + 金褐色配色
- 人體穴位圖、草藥圖示以現代扁平風格呈現
- 避免：舊式中醫書籍風格、過度裝飾性圖案

- **色彩對比：** 所有正文文字與背景對比率 ≥ 4.5:1

- **圖片 alt 文字：** 所有圖片必須有描述性 alt
- **表單標籤：** 每個輸入框都有關聯 `<label>`
- **語言宣告：** `<html lang="zh-Hant">`

- 輸入框 font-size 最小 16px（防止 iOS 自動縮放）
- 點擊目標最小 44px × 44px
